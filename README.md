# SecurePass - Password Generator

Proyecto final de **Laboratorio de Computacion II**.

Generador de contrasenas hecho con:

- HTML
- CSS
- JavaScript
- Canvas (medidor visual tipo semaforo)

## Funcionalidades

- Generacion de contrasenas aleatorias.
- Configuracion de longitud (slider).
- Seleccion de tipos de caracteres:
  - minusculas
  - mayusculas
  - numeros
  - simbolos
- Copia rapida al portapapeles.
- Estimacion de fuerza en bits + visualizacion en Canvas.

## Como ejecutarlo

### Opcion simple

Abrir `index.html` en el navegador.

### Opcion recomendada (Live Server)

1. Abrir la carpeta en VS Code.
2. Click derecho en `index.html`.
3. Seleccionar **Open with Live Server**.
